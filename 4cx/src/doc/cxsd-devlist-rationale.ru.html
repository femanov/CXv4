Резоны выбора формата файлов devlist

Файл "описания аппаратуры" должен решать две различные задачи.

  • С одной стороны, CX-серверу, работающему с управляющей аппаратурой, требуется список устройств.

    Этот список должен включать:

    • имена -- для адресации клиентов к каналам этих устройств;
    • типы устройств -- для выбора драйверов;
    • адресную информацию -- для доступа драйверов к устройствам.
  • Для клиентской же стороны нужно описание каналов УСТАНОВКИ -- токов, полей, и т.д., а также взаимосвязей между ними.

Эти две задачи тесно связаны и должны решаться совместно.
