Библиотека cxscheduler — руководство программиста

Введение

Библиотека-планировщик cxscheduler предоставляет основной цикл для программ, управляемых внешними событиями, на базе системного вызова select(). В качестве внешних событий могут выступать готовность файловых дескрипторов (на чтение, на запись, exception — в любых комбинациях), временные интервалы (таймауты) и сигналы.
